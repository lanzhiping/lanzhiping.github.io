## Project

This project involved contributing to the development of Burberry's e-commerce web application, which serves as a critical platform for the brand's online retail presence. My role encompassed frontend development, performance optimization, and implementing server-side rendering to enhance user experience and SEO.

## Details

Key contributions included frontend development for the e-commerce experience, performance improvements through server-side rendering and image optimization, and team enablement through JavaScript fundamentals training, onboarding support, and interviews.
