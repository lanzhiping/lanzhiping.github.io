## Context

In the era of AI, everyone is talking about how to build an all-in-one chat web application like ChatGPT. However, that will be a huge breaking change for users and developers. Instead, I am more interested in how we can build AI-First UI components that can be easily integrated into existing applications, empowering traditional UI with AI capabilities without requiring a complete overhaul.

## Current Direction

The article will focus on a practical component-level approach rather than a full chat-based application model. The core idea is to make AI assistance available inside familiar workflows, close to the user's current task and context.

## Planned Expansion

The final version will cover component patterns, integration details, and practical guidance for adding AI capability to existing product surfaces.
