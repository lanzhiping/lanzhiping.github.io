## PartnerCentral

As a Senior Software Engineer at ExpediaGroup (April 2021–present), I reported to the Director of Engineering and worked closely with PMs and UX designers to shape product direction and system architecture. I acted as technical lead across multiple initiatives, mentored junior developers, organized coding training sessions, and participated in hiring processes. I also facilitated scrum rituals including sprint planning, backlog grooming, and retrospectives.

Key achievements include:

- **Custom charting library** — Built a charting library adopted across multiple teams, standardizing data visualization practices and eliminating duplication.
- **Dynamic metrics dashboard** — Delivered a metrics dashboard for Expedia PartnerCentral that improved visibility into key product performance indicators for internal stakeholders.
- **UI toolkit integration** — Led the integration of a new UI toolkit into legacy systems via a reusable NPM package adopted department-wide.

**Tech Stack:** TypeScript, React, Node.js, ECharts, Webpack, GraphQL, NPM, AWS Lambda, API Gateway, ECS, GitHub Actions, Jenkins, Spinnaker
